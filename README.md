### CodSoft Internship - Task 1: Number Guessing Game

A simple console-based game built in Java where the user guesses a random number between 1 and 100. This project was completed as part of my Java Development Internship at CodSoft.

* * *

### 🎯 Game Features

*   **Random Generation:** Generates a secret number between 1 and 100.
*   **User Feedback:** Tells the user if their guess is "Too High" or "Too Low".
*   **Limited Attempts:** Adds a challenge by restricting the number of guesses.
*   **Score Tracking:** Tracks the user's score over multiple rounds of play.

* * *

### 🛠️ Technologies Used

*   **Language:** Java
*   **Libraries:** `java.util.Scanner` (for user input) & `java.util.Random` (for number generation)
